Resampling note review 2026-06-30

CENTRAL ISSUE — folds are not the identifying moments (now written into tex,
sec:identification_rank):
- "p params -> P=p folds -> p moments" is wrong. Under within-vintage
  resampling the estimation folds are EXCHANGEABLE subsamples of the same
  funds, so they share one population moment m-bar(theta). Stacked Jacobian
  = 1_P (d m-bar/d theta'), rank <= 1. Rank condition rank(dG/dtheta')=p
  fails for p>=2 no matter how many folds.
- Identifying rank lives in the CROSS-VINTAGE Jacobian J = d(g_v)/dtheta'
  (|A| x p), because vintages are NOT exchangeable (different factor paths).
  Scalar collapse replaces J by its column sum 1'J -> destroys the rank.
- Resolution (keep within-vintage design): identification comes from the
  within-fold vintage / strategy-vintage moment VECTOR; folds are only the
  holdout split. P=p_max is a comparability convention, not identification.
- Diagnostic: Jacobian rank/condition number at theta-hat, NOT moment count.
  Finite-sample "identification" is only cell-sampling noise -> weak ID a la
  Stock-Wright (2000). Report condition number + cross-split dispersion.

Does subagging/U-statistic idea fix this? NO. Bagging reduces variance/stabilizes
but cannot create population rank -> would give false precision (tight bag around
a pseudo-true point). Fix identification first (vector moments), THEN subagging
is a legitimate add-on (variance reduction + closed-form U-stat score).

4. [DRAFTED into tex, \subsection{Inference for score differences}, label

   draw). Plus multiplicity: report a model confidence set, not a single winner.
